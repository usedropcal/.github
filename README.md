# DropCal

**Send anything. Get a calendar invite.**

DropCal turns emails, photos, PDFs, and screenshots into calendar events. Forward anything to **add@usedropcal.com** and the invite comes back — no account, no signup, works with any calendar.

🪂 **[usedropcal.com](https://www.usedropcal.com)**

## How it works

1. Forward an email (or send a photo, PDF, or screenshot) to `add@usedropcal.com`
2. DropCal reads it and builds the event — date, time, location, attendees
3. You get a universal `.ics` invite back, or the event lands directly on your calendar if you've connected one

Dates, times, and timezones are resolved from context. Locations are geocoded. Multiple events in one message become multiple invites. Cc someone and they get the invite too.
